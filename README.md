## Proyecto SCSS Webmain

Framework SCSS modular basado en arquitectura 7-1 con namespace `wm-`, tokens de diseño, dark mode y componentes reutilizables.

### ✨ Optimizaciones realizadas
- Eliminación de imports innecesarios en `main.scss` (se eliminaron `vendors` y `themes`)
- Optimización de variables en `abstracts/_variables.scss` (nãming coherente, eliminación redundante)
- Consolidación de mixins en `abstracts/_mixins.scss`
- Limpieza de componentes (`_card.scss`, `_button.scss`, `_badge.scss`)
- Eliminación de utilidades duplicadas (`_bgx-color.scss`)
- Actualización de documentación y estructura final
